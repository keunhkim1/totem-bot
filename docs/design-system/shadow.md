# Shadow / Elevation

> **Note** — Toss 공식 문서에는 Shadow 토큰이 공개되어 있지 않습니다. 아래는 Toss 인터페이스에서 관찰되는 **3단계 엘리베이션 모범안**입니다.

---

## 스케일

| Token | Value | 용도 |
|---|---|---|
| shadow-subtle | `0 1px 2px rgba(0, 0, 0, 0.04)` | 카드 / 약한 분리 |
| shadow-medium | `0 4px 12px rgba(0, 0, 0, 0.08)` | 드롭다운 / 팝오버 |
| shadow-floating | `0 12px 24px rgba(0, 0, 0, 0.12)` | 모달 / 바텀시트 |

---

## 사용 가이드

- **카드**: subtle — 페이지 배경에서 살짝만 떠 있게.
- **메뉴 / 툴팁 / 팝오버**: medium.
- **모달 / 시트 / 다이얼로그**: floating + dim 오버레이(`greyOpacity500`).
- 그림자는 **표면이 떠 있다는 정보를 전달할 때만** 사용합니다. 단순 장식 금지.
- 어두운 배경에서는 그림자 대신 **보더(`grey200`)** 또는 **표면 색 차이**로 위계를 만듭니다.

---

## 안티패턴

- 모든 카드에 floating 수준의 짙은 그림자를 사용해 위계가 평탄해지는 경우.
- 그림자 + 짙은 보더를 동시에 사용해 시각 노이즈가 커지는 경우.
