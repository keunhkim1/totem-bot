# Radius

> **Note** — Toss 공식 문서에는 Radius 토큰이 공개되어 있지 않습니다. 아래는 Toss 인터페이스에서 관찰되는 **프로젝트 보충 모범안**입니다.

---

## 스케일

| Token | Value | 대표 용도 |
|---|---|---|
| radius-1 | 4px | 인풋 보더 / 마이크로 컴포넌트 |
| radius-2 | 8px | 칩 / 태그 / 작은 뱃지 |
| radius-3 | 12px | 버튼 표준 |
| radius-4 | 16px | 카드 표준 |
| radius-5 | 24px | 시트 / 모달 상단 |
| radius-full | 9999px | 원형 아바타 / pill 버튼 |

---

## 사용 가이드

- **버튼**: 12 — 손에 닿는 크기에서 가장 균형이 좋습니다.
- **카드**: 16 — 본문 컨테이너 표준.
- **바텀시트 / 모달 상단**: 24 — 시각적으로 떠 있는 표면을 강조.
- **태그/칩**: 8.
- **아바타**: full.
- 인풋과 버튼이 인접하면 **같은 라운드**(보통 12)로 맞춥니다.

---

## 안티패턴

- 한 화면에서 카드는 16, 모달은 12, 버튼은 8처럼 라운드가 거꾸로 가는 위계.
- 6, 10, 14 같은 비표준 라운드 도입.
