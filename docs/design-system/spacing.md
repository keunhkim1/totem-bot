# Spacing

> **Note** — Toss 공식 문서에는 Spacing 토큰이 공개되어 있지 않습니다. 아래는 Toss 인터페이스에서 관찰되는 **4/8 그리드 패턴 기반 프로젝트 보충 모범안**입니다.

---

## 스케일

기본 단위는 **4px**입니다. 8 이상은 **8 배수**를 우선 사용합니다.

| Token | Value | 대표 용도 |
|---|---|---|
| space-1 | 4px | 아이콘과 텍스트 사이 미세 간격 |
| space-2 | 8px | 같은 그룹 내 항목 간격 |
| space-3 | 12px | 라벨-입력 간격 |
| space-4 | 16px | 컴포넌트 내부 padding 표준 |
| space-5 | 20px | 카드 내부 여백 |
| space-6 | 24px | 카드 사이 / 그룹 사이 |
| space-8 | 32px | 큰 그룹 분리 |
| space-10 | 40px | 섹션 사이 |
| space-12 | 48px | 페이지 섹션 분리(대) |
| space-16 | 64px | 메이저 영역 분리 |

---

## 사용 가이드

- **컴포넌트 내부 padding**: 16 (표준), 작은 칩/뱃지는 8.
- **그룹 내부**: 항목 간 8.
- **그룹 간**: 24.
- **섹션 간**: 40~48.
- 4와 12는 마이크로 조정용이며, 12를 16 자리에 광범위하게 쓰지 않습니다.
- 6, 10, 14, 18 등 **그리드를 깨는 값은 사용하지 않습니다**.

---

## 안티패턴

- 한 화면에서 그룹 간격이 16/20/24/28로 들쭉날쭉한 경우.
- padding을 px 단위로 임의 입력해 스케일이 분산되는 경우.
